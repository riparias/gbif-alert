# API

The HTTP API is documented in-app at **`/api-docs`**, with the full interactive
reference for the current JSON API at **`/api/v2/docs`** (OpenAPI).

- **API v2 (recommended):** `/api/v2/` - interactive docs at `/api/v2/docs`.
- **WFS (GIS tools):** `/api/wfs/observations`.
- **Legacy `/api/` JSON endpoints:** deprecated and superseded by API v2; they
  will be removed in a future release. New integrations should use API v2.

> This file previously contained hand-maintained endpoint docs. They are now
> generated from the code and served at `/api/v2/docs`, so this pointer replaces
> them to avoid a second source of truth that drifts.
